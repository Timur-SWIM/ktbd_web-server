# Руководство пользователя

## Вход

Откройте `/login` и войдите под выданной учетной записью.

Демо-учетная запись:

- логин: `admin`
- пароль: `password`

## Панель управления

Dashboard показывает:

- количество изделий;
- количество сотрудников;
- количество документов;
- количество доступного оборудования;
- количество оборудования в работе.

## Работа с разделами

Доступные разделы зависят от роли пользователя:

- `engineer`: оборудование и изделия;
- `technologist`: элементы;
- `manager`: персонал;
- `director`: документы;
- `admin`: все разделы.

В каждом разделе доступны список, создание, редактирование, удаление и PDF-отчет.

## Изделия

В карточке изделия можно привязать:

- оборудование;
- элементы;
- документы.

Связи сохраняются в таблицах `device_tools`, `device_elements`, `device_documents`.

## PDF

Кнопка `PDF` в списке формирует отчет по таблице. В разделе документов отдельная кнопка `PDF` формирует файл по конкретному документу.
