# Руководство пользователя

## Вход

Откройте `/login` и войдите под выданной учетной записью.

Демо-учетная запись:

- логин: `admin`
- пароль: `password`

## Панель управления

Dashboard показывает:

- количество изделий;
- количество сотрудников;
- количество документов;
- количество доступного оборудования;
- количество оборудования в работе.

## Работа с разделами

Доступные разделы зависят от роли пользователя:

- `engineer`: оборудование и изделия;
- `technologist`: элементы;
- `manager`: персонал;
- `director`: документы;
- `admin`: все разделы.

В каждом разделе доступны список, создание, редактирование, удаление и PDF-отчет.

## Импорт из файлов

В разделах `Оборудование`, `Элементы` и `Изделия` кнопка `Создать` предлагает два способа:

- `Вручную` — открывает обычную форму создания записи.
- `Из файла` — загружает таблицу `.txt` или `.xlsx` с компьютера.

Импорт добавляет только новые записи. Если хотя бы одна строка содержит ошибку или дублирует уникальный номер, файл не сохраняется целиком.

Формат файла:

- первая строка содержит заголовки колонок;
- `.txt` использует разделитель `;`;
- `.xlsx` читается с первого листа;
- заголовки можно писать как системными именами полей, так и названиями из формы;
- для списков статусов можно указывать системное значение или русскую подпись.

Основные уникальные поля:

- оборудование: `Инвентарный номер` или `inventory_number`;
- элементы: `Артикул` или `part_number`;
- изделия: `Серийный номер` или `serial_number`.

Для изделий из файла импортируются только основные поля карточки. Связи с оборудованием, элементами и документами добавляются вручную после импорта.

## Изделия

В карточке изделия можно привязать:

- оборудование;
- элементы;
- документы.

Связи сохраняются в таблицах `device_tools`, `device_elements`, `device_documents`.

## PDF

Кнопка `PDF` в списке формирует отчет по таблице. В разделе документов отдельная кнопка `PDF` формирует файл по конкретному документу.
