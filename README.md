# CRM / АСУ ТП для производства электроники

MVP веб-приложения для учета производственных данных, персонала, оборудования, элементов, документов и изделий.

## Стек

- Oracle XE 21 в Docker
- PHP 8.2 + Apache
- OCI8
- Server-side rendering на PHP
- Bootstrap 5, DataTables
- TCPDF для PDF-отчетов с fallback-генератором PDF

## Быстрый запуск

1. Скопируйте `.env.example` в `.env` и при необходимости измените пароли.
2. Запустите проект:

```bash
docker compose up --build
```

3. Откройте приложение: `http://localhost:8090`.
4. Демо-доступ: `admin` / `password`.

## Основные возможности

- Авторизация и logout.
- Dashboard с производственными показателями.
- CRUD для оборудования, персонала, элементов, документов и изделий.
- Ролевой доступ на уровне UI и backend routes.
- PDF-отчеты по таблицам и отдельным документам.
- Автоматическая инициализация Oracle-схемы через SQL-скрипты.

## Проверка

После запуска контейнеров выполните:

```bash
./scripts/smoke.sh
```

Ручные сценарии приемки описаны в `TEST_PLAN.md`.
